## Full description (4000 chars max)

Muster is a free army-planning companion for miniature wargamers.

**Import your game data — nothing is bundled.**
Muster ships completely empty. Tap Import to fetch the latest community-maintained data files for your game system from BSData (github.com/BSData). No rules or unit data is stored inside the app — you're always in control of what you load.

**Browse factions and units.**
Once imported, explore every faction and unit your game system supports. Each unit card shows the full stat block, abilities, and keywords — everything you need to evaluate choices for your list.

**Build and manage army lists.**
Create multiple lists per faction, set a points limit, and add units directly from their detail pages. Muster tracks your running total and highlights when you've gone over budget.

**Key features:**
• Offline after first import — no internet needed to browse or build
• Unlimited army lists, each pinned to a faction with its own points cap
• Full stat blocks including all profiles and abilities from the imported data
• Points tracker with a live progress bar (turns red when over limit)
• Process-death safe — your list is still there after a force-stop
• Dark theme throughout

**Supported data format:** BattleScribe-compatible BSData files (.gst / .cat). Any game system published to BSData (github.com/BSData) works with Muster.

**Muster contains no game rules, no artwork, and no content owned by any game publisher.** It is a community data viewer — the imported data files are maintained independently by the BSData community.
